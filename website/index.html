Features




GitHub Integration

Self-Hosted

Run on your own server. Only requires nodejs, mysql and an email server.

Email Notifications

For every new release on a starred GitHub repository, you will receive a notification email.

Settings per Repo

You can enable or disable notifcations for each starred project individually.
